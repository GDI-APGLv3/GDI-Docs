# Tipos de Documento

El sistema incluye **61 tipos de documento** globales. Cada municipio habilita su propia seleccion a partir de este catalogo. Los tipos se dividen en **3 categorias** segun como se crea el contenido:

- **Documentos HTML** (41 tipos): se redactan en el editor de texto enriquecido del sistema.
- **Documentos importados** (18 tipos): se sube un archivo PDF externo.
- **Tipos internos** (2 tipos): los genera el sistema automaticamente y no estan disponibles en el selector.

!!! info "Disponibilidad por municipio"
    Cada municipio habilita su propia seleccion de tipos a partir de este catalogo global. Es posible que no veas todos los tipos listados aqui.

## Tipos internos (2 tipos)

Estos tipos **no estan disponibles en el selector** de creacion de documentos. El sistema los genera automaticamente cuando corresponde.

| Sigla | Nombre | Descripcion | Ejemplo de uso |
|-------|--------|-------------|----------------|
| PV | Pase | Se genera automaticamente al transferir un expediente | El sistema genera un PV al transferir |
| CAEX | Caratula | Se genera automaticamente al crear un expediente | El sistema genera un CAEX al crear expediente |

## Que tipo de documento debo usar?

Si no estas seguro de que tipo seleccionar, usa esta guia rapida:

| Necesito... | Tipo recomendado |
|-------------|------------------|
| Escribir un informe tecnico | **IF** (Informe) |
| Enviar una comunicacion a otro sector | **NOTA** (Nota) |
| Adjuntar un PDF externo (plano, informe escaneado) | **IFGRA** (Informe Grafico Importado) o **ANIMP** (Anexo Grafico Importado) |
| Crear un acto administrativo firmado por el intendente | **DECRE** (Decreto) |
| Registrar una inspeccion | **AINSP** (Acta de Inspeccion) |
| Dejar constancia de una reunion | **ACT** (Acta) o **MIN** (Minuta) |
| Emitir un permiso de obra | **POBRA** (Permiso de Obra) |
| Habilitar un comercio | **HCOM** (Cert. Habilit. Comercio) |
| Adjuntar una factura de proveedor | **FACT** (Factura / Remito) |
| Adjuntar un contrato firmado externamente | **CONT** (Contrato) |

## Preguntas frecuentes

??? question "Puedo ver todos los tipos del catalogo?"
    No necesariamente. Cada municipio habilita los tipos que necesita. Si un tipo no aparece en el selector al crear un documento, significa que tu municipio no lo tiene habilitado.

??? question "Cual es la diferencia entre un documento HTML y uno importado?"
    Un documento **HTML** se redacta directamente en el editor de texto de GDI. Un documento **importado** se crea subiendo un archivo PDF externo. Ambos pasan por el mismo proceso de firma.

??? question "Los tipos PV y CAEX aparecen en el selector?"
    No. Son tipos internos que el sistema genera automaticamente al transferir un expediente (PV) o al crear uno nuevo (CAEX). No se pueden crear manualmente.
