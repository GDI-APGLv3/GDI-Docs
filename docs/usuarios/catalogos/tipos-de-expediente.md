# Tipos de Expediente

El sistema incluye **30 tipos de expediente** globales. Cada organizacion habilita su propia seleccion a partir de este catalogo. El tipo de expediente define la **trata** (categoria del tramite) y aparece en el numero oficial del expediente.

!!! info "Disponibilidad por organizacion"
    Cada organizacion habilita su propia seleccion de tipos a partir de este catalogo global. Es posible que no veas todos los tipos listados aqui.

## Que tipo de expediente debo usar?

Si no estas seguro de que tipo seleccionar, usa esta guia rapida:

| Necesito tramitar... | Tipo recomendado |
|----------------------|------------------|
| Un asunto que no encaja en ninguna categoria | **EEVAR** (Varios) |
| Una compra directa de insumos | **COMP** (Compras y Contrataciones) |
| Una licitacion publica | **LICPUB** (Licitacion Publica) |
| Una habilitacion de local comercial | **HABI** (Habilitacion Comercial) |
| Un permiso para una fabrica | **HIND** (Permiso Industrial) |
| Una obra publica municipal | **OBPUB** (Obra Publica) |
| Un reclamo de un vecino | **RECVE** (Reclamo Vecinal) |
| Una solicitud interna de RRHH (licencia, vacaciones) | **SRRHH** (Solicitudes RRHH) |
| Un tramite de catastro o tierras | **CATIE** (Catastro y Tierras) |
| Una denuncia de defensa del consumidor | **DECON** (Defensa del Consumidor) |

## Preguntas frecuentes

??? question "Puedo cambiar el tipo de expediente despues de crearlo?"
    No. El tipo de expediente se define al momento de la creacion y no se puede cambiar despues. Por eso es importante elegir el tipo correcto desde el inicio.

??? question "Que hago si mi tramite no encaja en ningun tipo?"
    Usa el tipo **EEVAR (Varios)**. Es un tipo generico pensado para asuntos que no tienen una categoria especifica.

??? question "El tipo TEST es para uso real?"
    No. El tipo **TEST (Testing Automatizado)** esta reservado para pruebas del equipo de desarrollo. No debe usarse para tramites reales.
